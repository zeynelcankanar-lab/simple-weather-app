# Simple Weather App

Bu proje örnek bir açık kaynak uygulamasıdır. Herhangi bir API kullanmaz, örnek (fake) hava durumu verilerini gösterir.

## Özellikler
- Şehir ismi girildiğinde örnek hava durumu verir
- Çok basit Python kodu içerir
- Öğrenciler için açık kaynak proje örneği olarak uygundur

## Çalıştırma
Python kurulu ise terminalden:

python weather.py
# simple-weather-app
